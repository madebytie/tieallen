---
name: Writing style preferences
description: US English spelling, no em dashes, no "we/our" language - solo freelancer not a team
type: feedback
---

Use US English spelling (color not colour, favorite not favourite).

**Why:** User is US-based and noticed British spellings.

**How to apply:** Always use US English in copy for this project.

---

No em dashes (—) anywhere in copy. Use regular dashes (-) instead.

**Why:** User preference for cleaner punctuation.

**How to apply:** Never use em dashes in any content strings, metadata, or copy.

---

First person singular only - "I", "my", never "we", "our", "team".

**Why:** This is a solo freelancer portfolio site, not an agency.

**How to apply:** All copy across the site should read as one person speaking.
